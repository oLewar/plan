я поисковая система для страниц вики — на небольших масштабах достаточно индексного файла, но по мере роста вики вам потребуется полноценный поиск. [qmd](https://github.com/tobi/qmd) — хороший вариант: это локальная поисковая система для файлов Markdown с гибридным поиском BM25/векторным поиском и переранжированием LLM, всё на устройстве. Она имеет как интерфейс командной строки (чтобы LLM мог обращаться к ней через SSH), так и MCP-сервер (чтобы LLM мог использовать её как собственный инструмент). Вы также можете создать что-то попроще самостоятельно — LLM может помочь вам написать простой скрипт поиска на VibeCode по мере необходимости.
